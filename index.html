What is the Mandelbrot Set?

The Mandelbrot set is one of the most famous fractals in mathematics. It consists of all complex numbers c for which the function $z_{n+1} = z_n^2 + c$ does not diverge to infinity when starting with $z_0 = 0$.

$$M = \{c \in \mathbb{C} : |z_n| \text{ remains bounded as } n \to \infty\}$$

This set is particularly known for its incredible self-similarity and complexity, even at increasing magnifications. When zooming into an image of the Mandelbrot set, new fractal structures continuously emerge.

The Julia Sets

Julia sets are closely related to the Mandelbrot set. For each complex number c, there is a corresponding Julia set. The Julia set consists of all complex numbers $z_0$ whose orbit under iteration of $z_{n+1} = z_n^2 + c$ remains bounded.

$$J_c = \{z_0 \in \mathbb{C} : \text{the sequence } z_n \text{ remains bounded}\}$$

The Mandelbrot set serves as a map of Julia sets - points inside the Mandelbrot set correspond to connected Julia sets, while points outside correspond to disconnected "dust-like" Julia sets.

Escape Time Algorithm

The most common method for visualizing these sets is the escape time algorithm. For each point in the complex plane:

  1. We iterate the function $z_{n+1} = z_n^2 + c$ (starting with $z_0 = 0$ for the Mandelbrot set)
  2. We count how many iterations it takes before $|z_n| > 2$ (at which point we know it will escape to infinity)
  3. We assign a color based on this "escape time"

Points that never escape (or take more iterations than our maximum limit) are considered part of the set and typically colored black.
